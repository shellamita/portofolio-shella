Aplikasi Posyandu Digital – Desa Pangarengan

Aplikasi Posyandu Digital adalah sistem informasi manajemen kesehatan ibu dan anak yang dikembangkan berdasarkan penelitian lapangan di Desa Pangarengan, Kabupaten Cirebon. Proyek ini dirancang untuk meningkatkan efisiensi pencatatan data kesehatan, monitoring pertumbuhan balita, jadwal imunisasi, dan koordinasi antara kader posyandu dengan tenaga kesehatan.

Role

Full-Stack Developer & UX Researcher

Tech Stack

Laravel, MySQL, Figma, Miro, Javelin Board

Research & Development Methodology

Proyek ini menggunakan pendekatan Design Thinking dengan tahapan riset yang terstruktur untuk memastikan solusi yang dibangun benar-benar sesuai dengan kebutuhan pengguna di lapangan.

1 Penelitian Lapangan

Melakukan observasi dan wawancara mendalam dengan kader posyandu, bidan desa, dan orang tua balita di Desa Pangarengan untuk mengidentifikasi pain points dalam sistem pencatatan manual yang masih berjalan.

  • Wawancara dengan 5 kader posyandu aktif
  • Observasi proses pencatatan data manual selama 2 minggu
  • Analisis kendala koordinasi antara kader dan tenaga kesehatan
  • Dokumentasi workflow existing system

2 Javelin Board (Problem Validation)

Menyusun Javelin Board untuk memvalidasi asumsi masalah dan solusi. Framework ini membantu mengidentifikasi risiko tertinggi dalam pengembangan produk dan memprioritaskan validasi sebelum coding.

  • Customer Problem: Kesulitan tracking data balita secara konsisten
  • Solution: Sistem digital terintegrasi dengan reminder otomatis
  • Riskiest Assumption: Kader posyandu memiliki akses smartphone
  • Validation Method: Survey kepemilikan perangkat & tingkat literasi digital

3 Feature Mapping di Miro

Menggunakan Miro Board untuk brainstorming dan memetakan seluruh fitur aplikasi berdasarkan user journey dari kader posyandu, orang tua, dan tenaga kesehatan.

Fitur Utama:
  • Dashboard monitoring balita
  • Input & update data pertumbuhan
  • Jadwal imunisasi otomatis
  • Grafik tumbuh kembang (KMS digital)
  • Notifikasi reminder
Fitur Pendukung:
  • Laporan bulanan otomatis
  • Export data ke Excel/PDF
  • Role management (kader/bidan/admin)
  • Artikel kesehatan ibu & anak

4 Wireframe & Prototype (Figma)

Membuat wireframe low-fidelity untuk validasi alur pengguna, dilanjutkan dengan high-fidelity prototype untuk user testing dengan kader posyandu sebelum development.

  • Low-fidelity wireframe untuk 12 screen utama
  • High-fidelity prototype dengan branding warna kesehatan (hijau & biru)
  • Usability testing dengan 3 kader posyandu
  • Iterasi desain berdasarkan feedback lapangan

5 Development & Implementation

Mengembangkan aplikasi berbasis web menggunakan Laravel dengan database MySQL. Sistem dirancang mobile-responsive untuk aksesibilitas maksimal dari smartphone kader.

  • Backend API dengan Laravel 10
  • Frontend dengan Blade Template & TailwindCSS
  • Database normalisasi untuk efisiensi penyimpanan
  • Implementasi Chart.js untuk visualisasi data tumbuh kembang
  • Sistem notifikasi berbasis cron job

6 Hosting & Deployment

Melakukan deployment ke hosting dengan domain khusus untuk akses yang mudah diingat oleh pengguna di desa.

  • Hosting di shared hosting dengan cPanel
  • Setup domain: posyandu-pangarengan.com (contoh)
  • SSL certificate untuk keamanan data
  • Regular backup database mingguan
  • Training untuk kader dalam penggunaan sistem

Key Contributions

  • Melakukan riset lapangan dan analisis kebutuhan berbasis Design Thinking
  • Merancang product roadmap dengan Javelin Board untuk validasi risiko
  • Memfasilitasi sesi brainstorming fitur menggunakan Miro Board
  • Mendesain user interface yang sederhana dan mudah digunakan untuk pengguna dengan literasi digital terbatas
  • Mengembangkan full-stack application dari nol hingga deployment
  • Melakukan user acceptance testing (UAT) bersama stakeholder
  • Memberikan training dan dokumentasi penggunaan sistem

Impact & Results

  • Efisiensi waktu pencatatan meningkat 60% — dari manual ke digital
  • Akurasi data meningkat karena validasi otomatis dan mengurangi human error
  • Koordinasi lebih baik antara kader dan tenaga kesehatan melalui sistem terpusat
  • Reminder otomatis meningkatkan kehadiran imunisasi balita hingga 35%
